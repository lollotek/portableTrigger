Portable photo trigger

ITALIANO:
Software per attiny45 (dovrebbe essere compatibile anche per attiny85), per la realizzazione di un mini intervallometro.
L'intervallometro può essere regolato tramite un potenziometro, ed è utilizzabile con il sensore di luminosità oppure ad intervalli di tempo.
